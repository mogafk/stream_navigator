### Stream navigator

Слушает чат твича и читает все сообщения пока не находит
- сообщение `!180` делает разворот на `turnDistance` из файла `config.json`
- сообщение `!стан` блокирует управление клавиатурой(но не мышкой) на время указанное в значени `debugTime` в `config.json`

### Config

```
{
    
  // Команда для 180 сработает раз в 1m не чаще
  "cooldown180": "30s",
  // Кнопка, которая поворачивает камеру. Можно оставить пустым
  "debug180Key": "F3",
  // Сила поворат мыши, из-за разного DPI проще подобрать на глаз
  "turnDistance": 2000,
  
  // Команда для stun сработает раз в 1m не чаще
  "cooldownStun": "1m",
  // Кнопка, котоаря дает стан. Можно оставить пустым
  "debugStunKey": "F2",
  // Время стана
  "stunTime": "10s",

  // mute = true выключит звук при срабатывании команды
  "mute": false,

   // true - проигрывать музыкальные файлы, false - 
  "twitchLink": "https://www.twitch.tv/<your_name>"
}

```

Пустые значения `debug180Key` и `debugStunKey` просто выключают тригер этих событий по кнопке, команды из чата буду работать по прежнему.

### Сборка

```
go build -o stream_navigator64win.exe ./src/
```
